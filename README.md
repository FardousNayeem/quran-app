# Quran-Web

Quran-Web is a responsive Quran web application. It provides a clean reading experience with surah navigation, ayah display, translation, audio playback, search, and customizable font settings.

## Features

- Browse all 114 surahs
- View Arabic ayahs with English translation
- Surah details including name, number of ayahs, and revelation type
- Search ayahs by Arabic text or English translation
- Can jump to any Surah and any of its Ayah
- Play individual ayah recitation audio
- Adjustable Arabic and translation font sizes
- Multiple Arabic font options
- Font preferences saved using localStorage
- Dark theme UI
- Responsive layout for desktop, tablet, and mobile

## Tech Stack
- **Language:** TypeScript
- **Backend/API:** Hono-Bun
- **Frontend:** Next.js
- **Styling:** Tailwind CSS
- **Data Source:** quranapi (https://quranapi.pages.dev/api)
- **Audio Source:** quranapi (https://quranapi.pages.dev/api/audio/)
